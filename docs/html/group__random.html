◆ random_double()

double random_double ( )

Makes 64bit double from two 32bit random outputs.
Shift left one time is necessary because RAND_MAX=2147483647 and the highest bit is zero.

!!!won't work if random output is not 32bit or RAND_MAX!=2147483647!!!
